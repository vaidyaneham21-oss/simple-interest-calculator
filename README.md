# Simple Interest Calculator

This project is a basic Simple Interest Calculator.

## 📌 Formula
Simple Interest = (Principal × Rate × Time) / 100

## 📥 Inputs
- Principal amount
- Rate of interest
- Time period

## 📤 Output
- Calculated simple interest

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Enter principal amount
2. Enter rate of interest
3. Enter time
4. Click on calculate button
